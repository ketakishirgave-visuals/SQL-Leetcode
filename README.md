# SQL-Leetcode

this repository contains a curated set of sql problems solved from leetcode.
the focus is on writing clean, efficient, and interview-relevant sql queries.

## what this repo demonstrates
- strong understanding of joins, subqueries, and aggregations
- use of window functions like dense_rank, row_number, lag, lead
- handling real-world analytics problems such as rolling windows, rankings, and edge cases
- multiple solution approaches where applicable

## problem coverage
the problems include:
- salary and ranking problems
- department-wise analytics
- customer and order analysis
- time-based queries and rolling calculations
- self joins and conditional aggregations
- data validation using regex

each problem is stored as a separate `.sql` file with:
- a short explanation describing the logic
- clean, readable sql written in lowercase
- alternate solutions where relevant

## tools used
- sql (mysql )

this repository is meant to showcase sql fundamentals rather than volume.
